## Overview
This project is a Power BI dashboard that combines weather data and air quality metrics into a single, interactive view. It is designed for everyday usability ‚Äî helping users quickly understand current conditions, upcoming forecasts, and environmental quality.

Built as **Day 6 of my 7-Day Power BI Dashboard Challenge**.

---

## Problem Statement
Weather and air quality data often exist across multiple sources and formats, making it difficult to:
- Get a quick overview of current conditions
- Track upcoming temperature trends
- Understand air quality and pollutant levels together

This dashboard consolidates these metrics into a clean, user-friendly interface.

---

## Key KPIs & Metrics
- Current temperature and weather condition
- 7-day temperature forecast
- Sunrise & sunset time
- Humidity, wind speed, visibility, pressure
- UV index and precipitation
- Air Quality Index (AQI)
- Pollutant levels: PM2.5, PM10, CO, NO‚ÇÇ, SO‚ÇÇ

---

## Dashboard Features
- City-based comparison
- Line chart for 7-day temperature trend
- AQI indicator with pollutant breakdown
- Daily AQI comparison
- Minimal, modern UI focused on readability

- Designing dashboards for daily consumption
